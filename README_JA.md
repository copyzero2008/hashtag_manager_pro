# **📦 Hashtag Manager Pro**

**Hashtag Manager Pro**は、Twitter (X) のハッシュタグ管理と利用をより簡単にするために設計されたブラウザ拡張機能です。タグをカテゴリに整理したり、ドラッグ＆ドロップでプリセットを作成したり、独自の「優先順位ラダー（Priority Ladder）」機能を使って、重要なタグが文字数制限で切れないように管理したりできます。

![Main UI](images/hashtag_manager_pro_UI.png)

## **📖 開発の背景**

このツールは、私が **VRChat** コミュニティに参加する中で感じた個人的なニーズから生まれました。写真を投稿する際、適切なハッシュタグを追加したいと思っていましたが、毎回検索したりテキストファイルに保存しておいたものを探したりするのは面倒でした。そこで、自分で解決しようと決心しました。

このツールの目的は、面倒な作業を自動化し、コミュニティとのコンテンツ共有をより簡単にすることです。

## **✨ 主な機能**

* **📂 ダイナミックカテゴリ:** ハッシュタグをカスタムタブ（例：「アバター」、「ショップ」、「一般」など）に整理できます。  
* **🖱️ ドラッグ＆ドロップ UI:** カテゴリ間でハッシュタグを移動したり、Twitter/X のテキスト入力欄に直接ドラッグしたりできます。  
* **🪜 優先順位ソート:** 独自の並び順を定義できます。複数のタグをコピーする際、優先度の高いものが先に追加されるため、文字数制限で重要なタグが切れるのを防げます。  
* **📦 スマートプリセット:** よく使うハッシュタグの組み合わせ（例：特定のアバター「メグ」用など）を保存し、ワンクリックで展開できます。  
* **🌍 多言語サポート:** 英語、日本語、韓国語、簡体字中国語、繁体字中国語に対応しています。  
* **🎨 カスタマイズ:** ガラスモーフィズム（Glassmorphism）を採用したデザインで、自動、ライト、ダークテーマから選べます。  
* **🔒 プライバシー重視:** すべてのデータはブラウザの Chrome Storage API を介してローカルに保存されます。

## **🚀 使い方**

### **基本的な使い方**

1. **開く:** X.com (Twitter) にアクセスし、拡張機能のアイコンをクリックします。  
2. **カテゴリ作成:** タブリストの横にある \+ アイコンをクリックして、新しいカテゴリ（タブ）を作成します。  
3. **タグの追加:** タグリストの最後にある **\+ 追加** チップをクリックします。スペースで区切ることで、一度に複数のタグを追加できます。  
4. **コピー:** タグのチップをクリックするだけで、クリップボードにコピーされます。

### **複数選択モード**

チェックマークアイコンを使って **選択モード** に切り替えます。異なるカテゴリにまたがってハッシュタグを選択できます。

* **すべてコピー:** 下部に表示されるフローティングバーの「コピー」をクリックすると、選択したすべてのタグが一度にコピーされます。  
* **文字数カウンター:** 下部のカウンターは、タグが消費する文字数を示します。  
  * *注意:* 日本語、中国語、韓国語などの文字は2文字分としてカウントされます。

### **優先順位ソート ("ラダー")**

左側のサイドバーにある **優先順位ドロワー** を開いて、どのタグが重要かを定義します。

1. **追加:** タグまたはカテゴリごとラダーにドラッグします。  
2. **並べ替え:** アイテムを上下にドラッグして優先順位を変更します。  
3. **ロジック:** ハッシュタグとそのタグが含まれるカテゴリの両方をラダーに追加した場合、上位にある方が優先されます。  
4. **切り替え:** ドロワー内のトグルスイッチで、ソートロジックのオン/オフを切り替えられます。

### **プリセット**

**プリセットドロワー** を開いてバンドルを管理します。

* **作成:** \+ ボタンをクリックします。タグを入力するか、既存のチップを作成ボックスにドラッグできます。  
* **使用:** プリセットをクリックしてコピーするか、テキストフィールドにドラッグしてすぐに貼り付けられます。
