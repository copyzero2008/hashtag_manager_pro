# **📦 Hashtag Manager Pro**

**Hashtag Manager Pro**는 Twitter (X) 해시태그를 더 간단하게 관리하고 사용할 수 있도록 설계된 브라우저 확장 프로그램입니다. 태그를 카테고리로 정리하고, 드래그 앤 드롭으로 프리셋을 생성하며, 고유한 "우선순위 사다리(Priority Ladder)"를 사용하여 가장 중요한 태그가 글자 수 제한 내에 항상 포함되도록 할 수 있습니다.

![Main UI](images/hashtag_manager_pro_UI.png)

## **📖 개발 배경**

이 도구는 제가 **VRChat** 커뮤니티에 입문하면서 느낀 개인적인 필요에 의해 만들어졌습니다. 사진을 게시할 때 적절한 해시태그를 추가하고 싶었지만, 매번 검색하거나 텍스트 파일에 저장해 둔 것을 찾아보는 것이 번거로웠습니다. 그래서 직접 해결하기로 결심했습니다.

이 도구의 목표는 번거로운 작업을 자동화하여 커뮤니티와 콘텐츠를 더 쉽게 공유할 수 있도록 하는 것입니다.

## **✨ 주요 기능**

* **📂 동적 카테고리:** 해시태그를 사용자 지정 탭(예: "아바타", "상점", "일반")으로 정리할 수 있습니다.  
* **🖱️ 드래그 앤 드롭 UI:** 카테고리 간에 해시태그를 이동하거나 Twitter/X 텍스트 입력 필드로 직접 드래그할 수 있습니다.  
* **🪜 우선순위 정렬:** 사용자 지정 정렬 순서를 정의할 수 있습니다. 여러 태그를 복사할 때 우선순위가 높은 태그가 먼저 추가되어 글자 수 제한으로 인해 잘리는 것을 방지합니다.  
* **📦 스마트 프리셋:** 자주 사용하는 해시태그 조합(예: 특정 아바타 "Megu"용)을 저장하고 한 번의 클릭으로 불러올 수 있습니다.  
* **🌍 다국어 지원:** 영어, 일본어(日本語), 한국어, 중국어 간체(简体中文), 중국어 번체(繁體中文)를 지원합니다.  
* **🎨 사용자 지정:** 글래스모피즘(Glassmorphism) 스타일의 자동, 라이트, 다크 테마를 제공합니다.  
* **🔒 개인정보 보호:** 모든 데이터는 Chrome Storage API를 통해 브라우저 로컬에 저장됩니다.

## **🚀 사용 방법**

### **기본 사용법**

1. **열기:** X.com (Twitter)으로 이동하여 확장 프로그램 아이콘을 클릭합니다.  
2. **카테고리 생성:** 탭 목록 옆의 \+ 아이콘을 클릭하여 새 카테고리(탭)를 만듭니다.  
3. **태그 추가:** 태그 목록 끝에 있는 **\+ 추가** 칩을 클릭합니다. 공백으로 구분하여 여러 태그를 한 번에 추가할 수 있습니다.  
4. **복사:** 태그 칩을 클릭하기만 하면 클립보드에 복사됩니다.

### **다중 선택 모드**

체크표시 아이콘을 사용하여 **선택 모드**로 전환합니다. 여러 카테고리에 걸쳐 해시태그를 선택할 수 있습니다.

* **모두 복사:** 하단에 떠 있는 "복사" 바를 클릭하면 선택된 모든 태그가 한 번에 복사됩니다.  
* **글자 수 카운터:** 하단의 카운터는 태그가 차지하는 공간을 보여줍니다.  
  * *참고:* 일본어, 중국어, 한국어 문자는 2칸으로 계산됩니다.

### **우선순위 정렬 ("사다리")**

왼쪽 사이드바의 **우선순위 서랍**을 열어 어떤 태그가 가장 중요한지 정의합니다.

1. **추가:** 태그나 카테고리 전체를 사다리로 드래그합니다.  
2. **정렬:** 항목을 위아래로 드래그하여 우선순위를 변경합니다.  
3. **로직:** 해시태그와 그 태그가 속한 카테고리를 모두 사다리에 추가하면, 더 위에 있는 항목이 우선합니다.  
4. **전환:** 서랍 안의 토글 스위치로 정렬 로직을 켜고 끌 수 있습니다.

### **프리셋**

**프리셋 서랍**을 열어 번들을 관리합니다.

* **생성:** \+ 버튼을 클릭합니다. 태그를 입력하거나 기존 칩을 생성 상자로 드래그할 수 있습니다.  
* **사용:** 프리셋을 클릭하여 복사하거나 텍스트 필드로 드래그하여 즉시 붙여넣을 수 있습니다.
