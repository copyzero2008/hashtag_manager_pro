# **📦 Hashtag Manager Pro**

**Hashtag Manager Pro** 是一款旨在简化 Twitter (X) 标签管理和使用的浏览器扩展。它可以将标签整理到分类中，创建拖放式预设，并使用独特的“优先梯队（Priority Ladder）”功能，确保最重要的标签始终包含在字数限制内。

![Main UI](images/hashtag_manager_pro_UI.png)

## **📖 开发背景**

这个工具是出于我进入 **VRChat** 社区时的个人需求而创建的。在发布照片时，我想添加合适的标签，但总是查找或保存在文本文件中非常麻烦。因此，我决定自己动手解决这个问题。

目标是提供一个工具来自动化繁琐的流程，从而更容易与社区分享内容。

## **✨ 主要功能**

* **📂 动态分类:** 将标签整理到自定义标签页中（例如：“模型”、“摄影”、“活动”）。  
* **🖱️ 拖放界面:** 在分类之间移动标签，或直接将其拖入 Twitter/X 文本框。  
* **🪜 优先排序:** 定义自定义排序顺序。复制多个标签时，高优先级的标签会首先添加，防止因字数限制被截断。  
* **📦 智能预设:** 保存常用的标签组合（例如：针对特定角色“Megu”），并一键解包使用。  
* **🌍 多语言支持:** 支持英语、日语、韩语、简体中文和繁体中文。  
* **🎨 个性化:** 提供自动、亮色和暗色主题，采用玻璃拟态（Glassmorphism）设计。  
* **🔒 隐私专注:** 所有数据通过 Chrome Storage API 本地存储在浏览器中。

## **📥 安装方法**

*注意：此扩展未在 Chrome 网上应用店上架，需要手动安装。*

1. **下载:** 前往 \[suspicious link removed\] 页面，下载 Source code (zip) 或发布的 .zip 文件。  
2. **解压:** 将文件解压到计算机上的文件夹中。  
3. **打开扩展程序:**  
   * **Chrome/Brave:** chrome://extensions  
   * **Edge:** edge://extensions  
4. **启用开发者模式:** 打开右上角的开关。  
5. **加载已解压的扩展程序:** 点击按钮并选择刚刚解压的文件夹。  
6. **固定:** 点击浏览器工具栏中的拼图图标，将 **Hashtag Manager Pro** 固定。

## **🚀 使用方法**

### **基本用法**

1. **打开:** 前往 X.com (Twitter) 并点击扩展图标。  
2. **创建分类:** 点击标签页列表旁边的 \+ 图标创建新分类（标签页）。  
3. **添加标签:** 点击标签列表末尾的 **\+ 添加** 芯片。可以用空格分隔一次添加多个标签。  
4. **复制:** 只需点击任何标签芯片即可复制到剪贴板。

### **多选模式**

使用勾选图标切换到 **选择模式**。您可以跨不同分类选择标签。

* **全部复制:** 点击底部悬浮的“复制”栏，一次性复制所有选中的标签。  
* **字数计数器:** 底部的计数器显示标签占用的空间。  
  * *注意:* 日语、中文或韩语字符计为 2 个字符位。

### **优先排序 ("梯队")**

打开左侧的 **优先梯队抽屉**，定义哪些标签最重要。

1. **添加:** 将标签或整个分类拖入梯队。  
2. **排序:** 上下拖动项目以更改优先级。  
3. **逻辑:** 如果同时添加了标签及其所属分类，位置靠前的优先。  
4. **切换:** 可以使用抽屉内的开关开启/关闭排序逻辑。

### **预设**

打开 **预设抽屉** 管理组合。

* **创建:** 点击 \+ 按钮。您可以输入标签或将现有芯片拖入创建框。  
* **使用:** 点击预设进行复制，或将其拖入任何文本字段直接粘贴。

## **🛠️ 技术栈**

* **核心:** Vanilla JavaScript (无框架)  
* **清单:** Chrome Extension Manifest V3  
* **样式:** CSS (Glassmorphism)  
* **存储:** chrome.storage.local (数据持久化)
